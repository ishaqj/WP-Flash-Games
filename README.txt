== Description ==

WP Flash Games lets you build online arcade website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp_flash_games` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

NOTE: if Permalinks are not working then please go to Administration Panels > Settings > Permalinks, and press "Save".
